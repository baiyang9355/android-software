注意JDK是基于1.8的。1.8版本以下的并没有做过测试。

第一步：通过设置 File-->Settings-->Build,Execution,Deployment-->Compiler-->Java Compiler
中设置 Addition command paraters：为 -parameters

第二步：Build-->Rebuild Project重新build一下project就好了。
